Capstone Project - Data Science (Week 1)

 Project Overview
This project is a Week 1 capstone task focused on applying basic data science concepts such as data cleaning, exploratory data analysis (EDA), and visualization using Python.

 Dataset Used
Titanic dataset (real-world dataset used for analysis and learning purposes)

 Steps Performed
- Imported and loaded dataset
- Handled missing values and cleaned data
- Performed exploratory data analysis (EDA)
- Created visualizations using Matplotlib and Seaborn
- Extracted key insights from the data

Key Insights
- Majority of passengers were male
- Survival rate was influenced by gender and passenger class
- Age distribution showed many young passengers onboard

 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

 Outcome
Successfully completed a mini real-world data science project and gained hands-on experience in data preprocessing, visualization, and insight generation.
